# 开始你的第一个程序

1. 使用你自己的编辑器，写一个输出"Hello World"的程序。

```sql
display "Hello World"
```

2. 将你写的程序上传到服务器，并进行编译等一系列操作后，运行并测试是否能输出"Hello World"
